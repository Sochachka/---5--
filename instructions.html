📖 Инструкция

Как пользоваться нашим решателем уравнений
1
Введите уравнение – используйте переменную x. Примеры:
x^2 - 3x + 2 = 0
2x^3 + 4x = 0
x^2 + 1 = 0 (даст комплексные корни)
sin(x) = 0.5
tan(x) = 1
cos(x) = 0
2
Поддерживаемые функции – можно использовать в уравнениях:
sin(x) — синус
cos(x) — косинус
tan(x) или tg(x) — тангенс
cot(x) или ctg(x) — котангенс
ln(x) — натуральный логарифм
log(x) — десятичный логарифм
exp(x) — экспонента (e^x)
sqrt(x) — квадратный корень
abs(x) — модуль
π или pi — число Пи
e — число Эйлера
i — мнимая единица
3
Комплексные числа – пишите i (латинскую). Примеры:
(1+2i)x^2 + (3-4i)x + 5i = 0
x^2 + i = 0
sin(x) = 2i
4
Быстрый ввод – используйте клавиатуру под полем ввода:
• Цифры и основные операции: + - * / ^ ( )
• Константы: π e i
• Функции: sin cos tan ln exp sqrt abs
5
Нажмите «Решить» – результат появится в блоке "Ответ". Для тригонометрических уравнений выводятся красивые формулы:
• tan(x) = 1 → x = π/4 + π·k, k ∈ ℤ
• sin(x) = 0.5 → x = π/6 + 2π·k и x = 5π/6 + 2π·k
• cos(x) = 0 → x = ±π/2 + 2π·k
6
История – все ваши решения сохраняются в браузере. Вы можете очистить историю кнопкой внизу.
Важно:
  • Для полиномов до 5-й степени используются точные и численные методы
  • Тригонометрические уравнения решаются численно, но для простых случаев выводятся формулы
  • Если корней много, показываются только первые несколько с формулой периода
  • Все вычисления производятся на сервере с использованием метода Ньютона
